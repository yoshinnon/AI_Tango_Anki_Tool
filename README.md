# AI（Gemini）を利用した単語メモ＆サンプル文作成ツール

## 概要
学習中に見つけた英単語の「意味」「発音記号」「例文」「和訳」を、GoogleのAI（Gemini）が自動生成して登録する、語学学習効率化ツールです。
Streamlitベースの直感的なUIで、単語帳の作成からテスト、学習統計の確認までをワンストップで行えます。

## 主な機能（4つのモード）
本アプリには、学習をサポートするための以下の4つのモードが搭載されています。

1. **単語登録モード**
単語を入力するだけで、AIが詳細情報を生成。内容を確認・修正して保存できます。
また、複数の単語をカンマ区切りで入力し、一括でAI解析・保存を行います。
2. **単熟語テストモード**
登録したデッキを使って、4択形式のテストを実施します。
形式選択: 「英 → 日」「日 → 英」の切り替えが可能。
自動採点: テスト終了後、正誤一覧と解答時間を表示。満点時には演出（バルーン）があります。
3. **デッキ管理モード**
作成したデッキ内の単語検索、編集、削除。
不要になったデッキの完全削除。
4. **学習統計モード**
デッキごとのテスト履歴（スコア、正解率）を蓄積。
過去の学習成果を可視化します。

## 主な使用方法
基本的な学習のサイクルは以下の通りです。

1. **デッキの作成**
   まずは「デッキ管理モード」で、単語を登録するためのデッキ（単語グループ）を作成します。
2. **単語の登録**
   「単語登録モード」を使用し、単語を一つずつ、もしくは複数の単語を一括で登録します。
   - **【オプション機能】テーマ別自動一括登録**: テーマを設定することで、AIが自動で英単語リストを作成し、指定したデッキへの登録までを一括で行います（※1回の操作につき上限30単語までとなります）。
3. **テストの実施**
   「単熟語テストモード」にて、登録した単語を使用してテストを実施します。
4. **学習状況の確認**
   テスト結果は自動で採点され、「学習統計モード」に情報がまとめられます。日々の学習の進捗確認にお役立てください。

### 1. APIキーの準備
1. [Google AI Studio](https://aistudio.google.com/) でAPIキーを発行します。
2. 本アプリの実行ファイルと同じフォルダに、**`GOOGLE_API_KEY`** という名前のファイル（拡張子なし）を作成します。
3. 作成したファイルの中に、取得したAPIキーを貼り付けて保存してください。

### 2. Python環境の構築
### 1. インストーラーのダウンロード
1. [Python公式サイト (python.org)](https://www.python.org/downloads/windows/) にアクセスします。
2. 「Latest Python 3 Release」をクリックし、ページ下部の Files セクションから **Windows installer (64-bit)** をダウンロードします。

### 2. インストールの実行
1. ダウンロードした `.exe` ファイルをダブルクリックして実行します。
2. **重要：** インストール画面下部にある **「Add Python to PATH」** に必ずチェックを入れてください。
   - これにチェックを入れないと、ターミナルから `python` コマンドが使えなくなります。
3. 「Install Now」をクリックしてインストールを開始します。

### 3. インストールの確認
1. ターミナル（PowerShell または コマンドプロンプト）を開きます。
2. 以下のコマンドを入力し、バージョン番号が表示されるか確認します。

```
python --version
※ Python 3.x.x と表示されれば成功です。
```

以下のコマンドを実行し、必要なライブラリをインストールします。

```bash
pip install streamlit google-genai edge-tts pandas
```

Windowsの場合、pip環境にstreamlitをインストールしてもstreamlit.exeへのPATHが登録されていないため、`streamlit`コマンドがすぐには使えません。

```bash
>streamlit
'streamlit' は、内部コマンドまたは外部コマンド、
操作可能なプログラムまたはバッチ ファイルとして認識されていません。
```

そのため、streamlit.exeの場所をシステム環境変数（PATH）に登録します。
streamlitの場所を調べるために、アンインストールコマンドを実行してキャンセルします。

```
>pip uninstall streamlit
Found existing installation: streamlit 1.55.0
Uninstalling streamlit-1.55.0:
  Would remove:
    c:\users\＜ユーザー名＞\appdata\local\packages\pythonsoftwarefoundation.python.3.13_qbz5n2kfra8p0\localcache\local-packages\python313\scripts\streamlit.exe
  ~~~

Proceed (Y/n)? n
```
※上記の場合、PATHに新規登録するのは`c:\users\＜ユーザー名＞\appdata\local\packages\pythonsoftwarefoundation.python.3.13_qbz5n2kfra8p0\localcache\local-packages\python313\scripts`になります。

登録後、新規でターミナル（PowerShell または コマンドプロンプト）を開き、以下のコマンドを実行します。

```
streamlit hello
```

ブラウザが起動しでもアプリが表示されれば成功です（終了時はコマンドウインドウでCtrl + C）。


### 3. アプリの起動
ターミナル（PowerShell等）で本スクリプトのフォルダへ移動し、以下のコマンドを入力します。

```bash
cd ＜フォルダパス＞
streamlit run app.py
```

## 技術的な特徴

* **高精度な発音表示**: `Charis SIL` フォントを自動適用し、IPA発音記号を美しく表示します。
* **ネイティブ音声**: `edge-tts` を利用し、登録単語と例文の音声をその場で生成・再生します。
* **データ管理**: 登録データは `csv/` フォルダ内にデッキごとに保存され、Excel等での閲覧も可能です。

## 注意事項（2026/03/30 時点）

* **モデルの優先順位**: 無料枠を最大限活用するため、プログラム内で以下の順にAIモデルを切り替えて使用します（合計60回/日）。
    1. `gemini-3-flash-preview`
    2. `gemini-2.5-flash`
    3. `gemini-2.5-flash-lite`
* **おまけモデル**: 4番目の `gemini-3.1-flash-lite-preview` は、1日500回利用可能ですが、レスポンス速度が不安定なため補助的な位置づけとしています。
* **制限について**: 無料版Gemini APIは1モデルあたり20件/日の制限があります。より多くの解析が必要な場合は、有料版の契約を検討してください。
* **メンテナンス**: AIモデルの名称や仕様はGoogle側で随時更新されます。動作に不調を感じた場合は、同梱の `gemini_test.py` で接続確認を行うか、最新のモデル名にソースコードを書き換えてください。

```
python gemini_test.py
```
